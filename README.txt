SPREADSHEEP PROJECT  : 

- ce projet m'a plus sur  :
	- La rapidité d'intégration de la carte via Leaflet
	- Les multiples possibilités pour afficher les markers

- les taches difficiles ont été : 
	- Liason de la méthode layerChanged sur l'index

- Les améliorations :

	(Création du layer par l'utilisateur après la recherche de l'adresse)

	en fonction de sa propre géolocalisation.
